Hi There, Thanks for choosing my feedback plugin! Its super simple to get setup just follow the instrucions below for 0 issues

Just download the latest release and put the file in the plugins folder of the server

Next run the server once and go to the folder that it created and find your config.yml

Finally go to your discord server and make a channel with a webhook, Copy that webhook and place it in the area of the config.yml where it says (your Webhook here)

In the config.yml you may also change the amount of time needed to place another feedback message (default is 15 mins)! Thank you for using my plugin!

if there are any issues feel free to contact me via discord(Switchpro_) or the issues section of the plugin!
